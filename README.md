# entry-react

## elementの作成
### [構文] React.createElement(タグ名, 属性, 中に埋め込むもの)

## レンダリングした表示の作成
### [構文] ReactDOM.render(エレメント, DOM)

## Create React App
### [コマンド] npx create-react-app react_app

## 開発用のサーバプログラムを使ってプロジェクトを実行する
### [コマンド] npm start

## プロジェクトのビルドを行う。
### [コマンド] npm run build

## テストプログラムを実行し、アプリケーションのテストを行う
### [コマンド] npm test

## プロジェクトのイジェクトを行う
### [コマンド] npm run eject
